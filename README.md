# Socify (Swipeverse)

**Socify** is a UI prototype for an unbiased dating app built with
[Next.js (App Router)](https://nextjs.org/docs/app) and [Tailwind CSS](https://tailwindcss.com).
It demonstrates a swipe‑style profile view, responsive layout components and
iconography via [lucide-react](https://lucide.dev).

> 🚧 This repository currently contains static/sample data; the intent is to
> wire it up to a backend or API in future iterations.

---

## Features

- Landing page with animated call‑to‑action
- Responsive profile card (`HomeProfile` component) with attributes,
  image gallery and bottom info bar
- Mobile navigation component
- Custom icon wrappers for consistent sizing (using Lucide icons)
- ESLint + TypeScript configuration via Next.js defaults
- Deployed on Vercel (see “Deployment” section)
